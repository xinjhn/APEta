# Chapter Drafts

Folder ini berisi draft awal per bab yang mengikuti struktur template TA.
Draft ini belum menggantikan dokumen Word resmi. Gunakan sebagai sumber
konten untuk dipindahkan, disunting, dan diformat ke template.

Status awal:

- `BAB_I_PENDAHULUAN_DRAFT.md`: konteks masalah dan RQ awal.
- `BAB_III_METODOLOGI_DRAFT.md`: desain penelitian dua fase.
- `BAB_IV_APE_DRAFT.md`: dokumentasi APE sebagai aplikasi pendukung eksperimen.
- `BAB_V_HASIL_DAN_PEMBAHASAN_DRAFT.md`: cara menarasikan Phase 1 dan Phase 2.

Konvensi:

- Tanda `[Gambar ...]` berarti tempat gambar dari `FIGURE_REGISTER.md`.
- Tanda `[Tabel ...]` berarti tempat tabel dari `TABLE_REGISTER.md`.
- Tanda `TODO` berarti perlu data final, nama penulis, pembimbing, atau hasil
  eksperimen akhir.
