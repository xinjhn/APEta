# Phase 2 Migration Package

Paket ini dibuat untuk membantu mengubah draft `/home/ubuntu/APE/laporan_revised (2).docx` yang masih kuat bernuansa Phase 1 menjadi laporan yang selaras dengan eksperimen utama Phase 2.

Prinsip framing yang dipakai:

- Jangan menarasikan penelitian sebagai "gagal lalu pivot".
- Narasikan penelitian sebagai dua tahap yang wajar dalam eksperimen empiris:
  1. uji pendahuluan/pilot untuk kalibrasi instrument, validasi parity, dan membaca variabilitas awal;
  2. eksperimen utama Phase 2 dengan desain faktor yang lebih lengkap.
- Jangan mengklaim secara eksplisit bahwa semua keputusan Phase 2 sudah ada sejak awal kecuali ada bukti tertulisnya. Gunakan formulasi netral: "penelitian dilaksanakan dalam dua tahap".
- Phase 1 boleh muncul, tetapi hanya sebagai pilot/readiness experiment, bukan headline hasil utama.

File dalam folder ini:

- `DRAFT_AUDIT_PHASE2.md`: audit masalah pada draft saat ini.
- `SECTION_REWRITE_GUIDE.md`: panduan per bagian template untuk migrasi ke Phase 2.
- `PASTE_READY_REPLACEMENTS.md`: paragraf pengganti siap tempel.
- `SOURCE_OF_TRUTH_PHASE2.md`: ringkasan fakta eksperimen Phase 2 berdasarkan workspace.
